Explainable AI Recommendation System (XAI)

Overview
This project implements an Explainable AI (XAI)-based recommendation system using machine learning models. It focuses not only on prediction accuracy but also on making model decisions transparent using SHAP and LIME.

 Objectives
* Build accurate recommendation models using ML algorithms
* Improve model interpretability using Explainable AI techniques
* Help users understand *why* a prediction was made

 Tech Stack
* Python
* Scikit-learn
* SHAP
* LIME
* Pandas, NumPy
* Matplotlib, Seaborn

 Dataset
* Size: 10,000+ records
* Type: Structured dataset for recommendation/classification
* Features: Multiple input variables used for prediction

 Features
* Data preprocessing and cleaning
* Model training using:
  * Decision Tree
  * Random Forest
  * Neural Networks
* Model evaluation (accuracy, performance metrics)
* Explainable predictions using SHAP and LIME
* Visualisation of feature importance


 Results
* Achieved **85–90% accuracy** across models
* Improved model performance by **~12%** through tuning
* Generated interpretable outputs for each prediction

 Performance Improvements

* Reduced training time from **45 minutes → 30 minutes (~33% faster)**
* Optimised feature selection for better efficiency

---

 Output / Visualisations
* SHAP summary plots
* Feature importance graphs
* Model comparison charts
